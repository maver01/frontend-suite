# frontend-suite

This repository contains a series of modules that can be used to build a frontend.

Folder structure:

```
├── platform                        (e.g.: web)
│   └── framework                   (e.g.: react)
│   │   │   └──  project            (e.g.: web-send-message-v1)
│   │   │   │   └──  source code    (e.g.: src)
```
