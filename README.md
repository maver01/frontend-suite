# frontend-suite

This repository contains a series of modules that can be used to build a frontend.

Folder structure:

```
├── platform                        (e.g.: platform-web)
│   └── framework                   (e.g.: framework-react)
│   │   │   └──  project            (e.g.: project-web-send-message-v1)
│   │   │   │   └──  source code    (e.g.: src)
```
